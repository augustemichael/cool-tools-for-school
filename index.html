Cool Tools for School

This project is a simple HTML website created for Lesson 2: Building a Webpage Template. It shows how to use HTML to structure a webpage. The goal is to help students organize online study resources and simple tools for school.

Project Overview

The Cool Tools for School website gives students a basic layout for listing tools, links, or study materials. It uses basic HTML elements such as headings, paragraphs, and lists to organize content.

Built With

  • HTML5 for structure
  • CSS (optional, not used in this version)
  • Simple JavaScript to display the current year
