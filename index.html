Project Overview

This project is a simple HTML and CSS website created for Lesson 2: Building a Webpage Template. It shows how to use HTML to structure a webpage and CSS to add basic style and layout. The goal is to help students create and organize online resources for school.

Built With

This website uses the following:

  • HTML5 for structure
  • CSS3 for styling
  • JavaScript for a simple year update
